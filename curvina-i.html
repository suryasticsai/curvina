🧜‍♀️ What is CurVina?

CurVina is an AI‑powered resume tailoring tool that helps job seekers customize their resumes for specific job descriptions. It uses Retrieval-Augmented Generation (RAG) to extract the most relevant skills and experiences from your existing resume and rewrite them to match the target job, increasing your chances of passing Applicant Tracking Systems (ATS).

⚙️ How It Works

  1. Input Job Description – Paste a job URL, text, or upload a file.
  2. Upload Your Resume – Upload PDF, DOCX, TXT, or fetch from LinkedIn / Gist.
  3. Curify – Use the comparison panel to selectively regenerate sections (Summary, Roles, Skills, Hobbies) based on the job description.
  4. Edit & Export – Fine‑tune the suggestions and export as DOCX, PDF, HTML, or plain text.

✨ Advantages

  • ATS‑Optimized – Tailors content to include keywords from the job description.
  • Granular Control – Curify specific sections instead of the whole resume.
  • Time‑Saving – Automates the tedious process of customizing resumes for each application.
  • Privacy‑First – All processing happens locally in your browser (mock mode for demo).

🛠️ Tech Stack

HTML5 CSS3 JavaScript (ES6) Bootstrap 5 Font Awesome Canvas API Web Audio API RAGina.js (TF‑IDF) Mock LLM (simulated)

🚀 Future Roadmap
